Features

本地运行

无需服务器，不离开浏览器，直接在本地预览 HTML 文件及其所有依赖。打开文件夹，立刻看到页面。

多设备模拟

自动根据窗口尺寸切换手机、平板、桌面三种预览外壳。所见即所得，不用打开开发者工具。

资源自动注入

CSS、JS、图片、字体 — 页面引用的所有资源自动解析、内联加载。不需要配路径，打开就能看。

多页面切换

文件夹里有多个 HTML 文件？底部的标签栏让你一键切换，方便对比不同页面。
