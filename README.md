# Preview

给开发者用的本地 HTML 预览工具。选择文件夹，立即预览，不需要配置服务器。

## 用法

1. 打开 [preview](https://chibaqy.github.io/preview)
2. 点击「选择文件夹」或拖拽文件夹到页面
3. 自动加载文件夹内所有 HTML 文件
4. 侧边栏点文件切换预览，底部标签栏快速切换页面

## 特性

- **本地运行** — 完全在浏览器中运行，不需要服务器
- **资源自动注入** — CSS、JS、图片自动解析和加载
- **多设备模拟** — 根据窗口尺寸自动切换手机 / 平板 / 桌面外壳
- **多页面切换** — 底部标签栏一键切换文件夹内的 HTML 文件
- **深色模式** — 支持日间 / 夜间切换，自动跟随系统偏好

## 开发

```bash
git clone https://github.com/ChibaQY/preview.git
cd preview
# 不需要构建，直接打开 index.html 或 app.html
```
